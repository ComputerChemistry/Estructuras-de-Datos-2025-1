Detailed Description

template<typename Tipo>
class Pila< Tipo >

Implementación de una pila genérica basada en nodos.

Esta clase implementa una pila genérica usando nodos, soportando las operaciones básicas como apilar, desapilar y obtener el valor del tope de la pila. La pila también permite verificar si está vacía o llena.

Template Parameters
TipoEl tipo de datos que almacena la pila. Puede ser cualquier tipo de dato.
